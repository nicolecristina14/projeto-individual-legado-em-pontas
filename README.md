# 🩰✨ LEGADO EM PONTAS
> Onde cada passo conta uma história de dedicação e arte.

## 📌 Sobre o Projeto

O Legado em Pontas é uma plataforma dedicada à preservação e celebração da memória do ballet clássico no Brasil, estabelecendo uma ponte entre a tradição histórica e a modernidade tecnológica. O projeto nasce da premissa de que a dança, vivida entre sapatilhas e palcos, é uma linguagem que une disciplina e arte, traduzindo a trajetória de figuras fundamentais que construíram a base da dança nacional, desde as pioneiras até os grandes nomes da atualidade. Através de uma interface que prioriza a elegância e a fluidez, a plataforma oferece um acervo histórico detalhado, sistemas interativos de identificação de perfil baseados em lógica de programação e dashboards que transformam a dedicação artística em visualizações de dados funcionais. Desenvolvido com foco em uma experiência imersiva, o projeto utiliza tecnologias como HTML, CSS e JavaScript em ambiente Linux para garantir que o rigor técnico da computação honre a sensibilidade da dança, transformando a paixão pelo movimento em um registro vivo e inspirador para as futuras gerações.
