# 🎧👽 SET DOS DJ's

> Um universo onde a música conecta pessoas.

---

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um **site interativo no formato de blog sobre o universo DJ**, com foco nas preferências musicais e no comportamento do público em festas.

A ideia surgiu a partir da minha vivência como DJ, onde percebo como a música influencia diretamente a experiência das pessoas na pista de dança.

O sistema permitirá a interação dos usuários por meio de formulários e enquetes, gerando dados que serão exibidos em um dashboard.
